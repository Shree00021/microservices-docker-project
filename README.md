# Microservices Docker Project

## 📌 Description
This project is a simple microservices-based application built using Docker and Docker Compose.

## 🧱 Services
- User Service (Port 5001)
- Task Service (Port 5002)
- Frontend (Port 3000)

## ⚙️ Technologies Used
- Docker
- Docker Compose
- Python (Flask)
- HTML, JavaScript

## 🚀 How to Run

1. Clone the repository
2. Run:
   docker-compose up --build
3. Open:
   http://localhost:3000

## 📊 Features
- Add Users
- Add Tasks
- View Users and Tasks
